# 🎬 Netflix Dashboard Analysis Using Power BI

## 📌 Project Overview

This project focuses on analyzing Netflix's content library using **Microsoft Power BI** to uncover trends in movies and TV shows available on the platform. The objective is to transform raw Netflix dataset information into meaningful insights that help understand content distribution, audience ratings, release patterns, and genre popularity.

## 🎯 Objectives

* Analyze the distribution of Movies and TV Shows on Netflix.
* Identify content trends across different release years.
* Examine IMDb ratings and viewer votes.
* Understand content classification based on age certifications.
* Discover the most popular and highly-rated titles.
* Create interactive visualizations for data-driven decision-making.

## 📂 Dataset Description

The dataset contains detailed information about Netflix titles, including:

* Title Name
* Content Type (Movie/TV Show)
* Release Year
* Runtime
* IMDb Rating
* IMDb Votes
* Age Certification
* Genre Information

The dataset helps analyze content performance and viewing trends across different categories.

---

## 📈 Key Insights

* Netflix's content library is dominated by Movies compared to TV Shows.
* Significant growth in content releases occurred between 2015 and 2021.
* TV-MA and TV-14 are the most common age certification categories.
* Highly-rated titles tend to receive significantly more viewer votes.
* Recent years show increased content production, indicating Netflix's aggressive expansion strategy.
* Certain titles consistently achieve higher IMDb ratings and audience engagement.
